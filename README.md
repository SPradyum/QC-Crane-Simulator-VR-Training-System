# 🏗️ QC Crane Simulator – VR Training System

A VR-based training simulation for operating a Quay Crane (QC) used in container handling at ports. This project provides an interactive and immersive environment to practice crane operations safely and efficiently.

---

## 🚀 Features

- 🎮 Real-time interactive crane control (levers & buttons)
- 🏗️ Simulates container loading and unloading operations
- 🧭 Step-by-step guided training for new operators
- ⚠️ Safety-focused environment with zero real-world risk
- 🎯 Enhances precision, coordination, and operational efficiency

---

## ⚙️ How It Works

1. **Environment Setup** – Simulated port environment with crane model  
2. **User Interaction** – Controls mapped to crane movement (lift, rotate, move)  
3. **Simulation Engine** – Processes input and updates crane actions in real time  
4. **Training Flow** – Step-by-step instructions for guided learning  
5. **Feedback Loop** – Visual feedback for positioning and movement accuracy
